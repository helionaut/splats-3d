Gaussian Splats 3D Viewer

Welcome to the Gaussian Splats 3D Viewer project. This tool allows you to visualize 3D Gaussian splatting models with a user-friendly interface.

Launch 3D Viewer

About This Project

This viewer allows you to explore 3D scenes created with Gaussian Splatting technology. The viewer provides an interactive interface to navigate and examine 3D point cloud-like representations.

For local viewing:

  1. Clone this repository
  2. Navigate to the GaussianSplats3D directory
  3. Run npm run demo
  4. Access the viewer at http://127.0.0.1:8080/custom.html

This project uses mkkellogg's GaussianSplats3D implementation.
